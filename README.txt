PythieOfConcept
par Dimitri Watel

Requis: 
- java8, 
- php, avec le module mbstring activé
- un navigateur acceptant d'exécuter du javascript 

Optionnel : 
- python3 pour pouvoir lancer une des IA codée en python3

Lancement :
- extraire tous les fichiers dans un dossier (pour l'exemple, on l'appelera pythieofconcept)
- aller dans le dossier pythieofconcept



Sous Linux : 

Pour avoir de l'aide
- java -jar pythieofconcept.jar -h

Pour tester
- java -jar pythieofconcept.jar -b3 -b2 
- cd html
- php -S localhost:8888 &
- firefox index.php



Sous Windows ou MacOS:

Installez une distribution Linux fonctionnelle

ou

https://duckduckgo.com/?q=windows+execute+jar
https://duckduckgo.com/?q=macos+execute+jar
https://duckduckgo.com/?q=windows+run+php+server
https://duckduckgo.com/?q=macos+run+php+server
